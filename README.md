# automatic-adventure

> **Chapel of Anticipation** — A personal Python practice repository.

## Purpose

This repository is my dedicated space for practicing and strengthening **fundamental Python programming skills** as I prepare to begin the **Duke University Master of Engineering in Artificial Intelligence for Product Innovation (AIPI)** program.

While the program will involve more advanced AI/ML topics, I believe a strong foundation in Python is essential. This repo will serve as a living notebook and practice ground for core concepts before and during the program.

## Goals

- Build and reinforce strong Python fundamentals
- Practice clean code, structure, and documentation habits
- Create a reference library of examples I can revisit during the AIPI program
- Experiment with small scripts and exercises in a low-pressure environment
- Develop good repository and version control habits
